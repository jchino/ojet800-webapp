# Oracle JET v8.0.0 Sample Web Application

## Scaffold の生成

```sh
npx -p @oracle/ojet-cli -c "ojet create ${app_dir} --template=blank"
cd ${app_dir}
npm install @oracle/ojet-cli --save-dev
```
